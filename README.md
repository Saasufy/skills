# Skills
Saasufy AI Skills Marketplace

## Installing Saasufy Skills with Claude Code

### Add the Saasufy Skills Marketplace

After launching Claude Code, run this command:

```
/plugin marketplace add saasufy/skills
```

### Install the Saasufy Skill

After adding the Saasufy marketplace, search for the `saasufy` plugin:

```
/plugins
```

Browse to the `saasufy` plugin and install.

### Setup your Saasufy Application

Once you've installed the Saasufy plugin/skill, you can just ask Claude to set up your app with a prompt like:

```
Please set up this Saasufy app.
```
